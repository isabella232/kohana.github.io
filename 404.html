Sorry, the page you are looking for could not be found. The Kohana framework has been deprecated - for more information and links to alternatives see the homepage.

The userguide and API documentation browser is no longer available online. Guide pages in markdown format are available under the /guide path in each of the kohana module repositories. For example, core framework guides are are available here.

You will also find useful information within the phpdoc comments of all Kohana classes, modules and properties.

You can easily configure your local kohana installation to serve the full userguide and API docs browser. Follow the installation guide to set up a local installation of kohana, uncomment the userguide module in application/bootstrap.php and visit http://localhost/kohana/guide.
